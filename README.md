📊 Análisis de Movilidad Urbana y Economía en Ciudades Latinoamericanas
### 🎯 Descripción
Análisis de la relación entre congestión vehicular y productividad económica en ciudades latinoamericanas para identificar prioridades de inversión en infraestructura de transporte.

### 🔍 Pregunta Principal
¿Qué ciudades presentan alta congestión y baja productividad económica, requiriendo inversión prioritaria en movilidad urbana?

### 🛠️ Tecnologías
- Python 3.9+ | Pandas | NumPy | Seaborn | Matplotlib

### 📊 Fuentes de Datos
- TomTom Traffic Index: Datos de congestión vehicular
- OECD Cities: Indicadores económicos por ciudad

### 🔄 Metodología
1. Limpieza: Estandarización de columnas, conversión de tipos de datos
2. Procesamiento: Agregación de métricas por ciudad (promedios anuales 2024)
3. Análisis: Unión de datasets y análisis exploratorio
4. Visualización: Boxplots, histogramas y gráficos comparativos

### 📈 Principales Hallazgos
- No existe relación lineal clara entre PIB per cápita y congestión
- Ciudad de México: Mayor congestión promedio (outlier)
- Bogotá: Alta congestión + menor PIB → Ciudad prioritaria para inversión
- Buenos Aires: Mejor balance economía-movilidad
